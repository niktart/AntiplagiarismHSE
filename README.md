\# Микросервисный проект



\## Структура

\- `FileAnalysis/` — сервис анализа файлов

\- `FileStoring/` — сервис хранения файлов

\- `Gateway/` — API Gateway



\## Запуск

1\. Запустите каждый сервис отдельно через Docker или .NET CLI.

2\. Gateway выступает как единая точка входа.



\## Технологии

\- ASP.NET Core

\- Entity Framework Core

\- Docker

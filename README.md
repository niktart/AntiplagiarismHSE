# Антиплагиат - Система проверки учебных работ
# Тартышев Никита Геннадьевич БПИ248

Микросервисная система для загрузки, хранения и проверки студенческих работ на плагиат.

## Архитектура

Система состоит из 3 микросервисов:

### 1. **Gateway** (Порт: 8080)
- Единая точка входа для клиентов
- Маршрутизация запросов к соответствующим сервисам
- Обработка ошибок и логирование

### 2. **FileStoring** (Порт: 80 внутри сети)
- Хранение загруженных файлов в БД
- Управление метаданными работ (студент, задание, дата загрузки)
- Предоставление доступа к файлам по ID

### 3. **FileAnalysis** (Порт: 80 внутри сети)
- Анализ текстов на плагиат с использованием алгоритма шинглов
- Генерация отчетов и статистики
- Создание облака слов через QuickChart.io

## Технологический стек

- **Backend**: ASP.NET Core 8.0
- **База данных**:  Sqlite
- **Контейнеризация**: Docker + Docker Compose
- **Алгоритм плагиата**: Shingling (шинглы размером 3)

## Быстрый старт
docker compose up --build

## Как работать с программой
### POST /api/works/submit
загрузка работы студентом
### GET /api/works/{assignmentId}/reports
Получить все загруженные работы с результатом плагиата по assignmentId
### GET /api/works/submission/{submissionId}/report
Полный отчет по одной работе
### GET /api/works/{assignmentId}/submissions
Все загруженные работы на assignmentId
### GET /api/works/file/{submissionId}
Получить и скачать загруженную работу
